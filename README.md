# Stud_projects
Data Science &amp; Machine Learning study projects | LETI University

Добро пожаловать в мой репозиторий с учебными проектами! Здесь собраны практические работы, выполненные мной в рамках обучения в СПбГЭТУ «ЛЭТИ» (направление ПМИ, профиль «Системы ИИ»)

## Основные проекты

### 1. Исследовательский анализ данных (PCA и Кластеризация)
*   **Задача:** Снижение размерности многомерных данных и поиск скрытых закономерностей.
*   **Что сделано:** Использовала метод PCA для визуализации данных, провела кластерный анализ для разделения объектов на группы.
*   **Результат:** Построены графики распределения признаков, определено оптимальное количество кластеров.

### 2. Обработка текста (NLP)
*   **Задача:** Препроцессинг текста и создание языковой модели.
*   **Что сделано:** Очистка текста от шума, токенизация, обучение эмбеддингов Word2Vec и построение предсказательной модели на базе LSTM.
*   **Результат:** Модель предсказывает следующее слово в последовательности на основе контекста.

### 3. Компьютерное зрение (CNN и VAE)
*   **Задача:** Классификация изображений и генерация новых данных.
*   **Что сделано:** Обучила сверточную сеть (CNN) на датасете CIFAR-10. Реализовала вариационный автоэнкодер (VAE) для создания синтетических изображений.

### 4. Обучение с подкреплением (Reinforcement Learning)
*   **Задача:** Обучение агента стратегии игры в BlackJack.
*   **Инструменты:** Gymnasium, алгоритм Actor-Critic.
*   **Результат:** Обученный агент, принимающий решения на основе Value-функции.
